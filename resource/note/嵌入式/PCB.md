#### 三极管
三极管结构

+ **B（Base）基极** → 控制端（**B 脚一定要串电阻**（限流）  ）
+ **C（Collector）集电极** → 接负载
+ **E（Emitter）发射极** → 接地或电源

 基极有电流 → 集电极和发射极“导通”（NPN）（PNP导通条件相反）

## 元件符号设计
1. 文件——新建——元件
2. 命名元件
3. 去立创商城搜元件数据手册，照着手册上的绘制形状
4. 点击引脚，右边菜单栏修改引脚名称、编号、长度等效信息
5. 在1号引脚位置画上小圆圈

## 元件封装绘制
1. 文件——新建——封装
2. 命名元件
3. 去立创商城搜元件手册，找到关于尺寸那一张，依据手册画焊盘，但一般焊盘尺寸要弄得稍微大一点，但相对位置如间距应该要一样

3. 最后处理GND铺铜
    1. 弄一个过孔阵列
    2. 在元件GND附近和空旷区域放满这些过孔
    3. 工具栏选择铺铜——框线整个板子——选择顶层——确认，再选择底层重复一次
4. 左侧工程设计看飞线数量是否为0，为0则表示全部连接上了
5. （若还存在大量空白区域，可选择在这些空白区域继续添加过孔，然后右键铺铜——铺铜区域——重建所有
